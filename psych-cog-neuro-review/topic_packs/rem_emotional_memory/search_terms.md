# REM 与情绪记忆主题包：检索词 / Search Terms

## 1. 主题边界

本主题包适用于以下方向：

- REM sleep 与 emotional memory
- REM physiology 与 affective processing
- sleep-dependent emotional memory consolidation
- fear conditioning, extinction, extinction recall, reconsolidation
- PSG、EEG/ERP、fMRI 与行为指标整合

不要把以下主题直接等同于 REM 与情绪记忆：

- 一般睡眠质量与一般记忆
- 梦内容研究但无记忆任务
- 情绪调节但无睡眠或记忆过程
- 仅失眠或睡眠障碍流行病学且无情绪记忆指标

## 5. 检索记录提醒

- 单独记录 REM、NREM、整夜睡眠、小睡、睡眠剥夺的检索结果。
- 单独记录 emotional memory、fear conditioning、extinction、reconsolidation 的检索结果。
- 不要因为检索式含 REM 就默认所有睡眠研究都能支持 REM 机制结论。
- 中文数据库检索需分别记录 CNKI、万方、维普。
