# REM 与情绪记忆主题包：机制框架

## 1. 核心机制问题

本主题包默认围绕以下问题组织证据：

1. REM sleep 是否与情绪记忆巩固有关？
2. 这种关系是否特异于负性、正性或高唤醒材料？
3. REM 指标是否能解释恐惧条件反射、消退学习、消退回忆或再巩固？
4. 行为证据、PSG、EEG/ERP 和 fMRI 是否指向一致机制？
5. REM 相关发现是否可以推广到临床或高风险样本？

## 2. 证据层级

| 层级 | 证据类型 | 可支持的写法 | 不可支持的写法 |
|---|---|---|---|
| 直接证据 | 睡眠操控 + 记忆任务 + PSG/REM 指标 | REM 相关睡眠过程与某类记忆表现相关/可能参与 | REM 直接决定情绪记忆 |
| 中等证据 | 自然睡眠差异 + 记忆任务 | REM 指标与记忆表现存在关联 | REM 造成记忆变化 |
| 间接证据 | EEG/fMRI 情绪任务但无睡眠操控 | 相关神经系统可能参与情绪记忆加工 | 证明 REM 机制 |
| 背景证据 | 综述/理论论文 | 用于理论框架和研究脉络 | 冒充原始研究结果 |

## 3. 推荐综合结构

### 3.1 行为层面

- 记忆再认、回忆、辨别力、反应时、d′、bias
- 正性、负性、中性材料差异
- 任务延迟时间与睡眠时机

### 3.2 睡眠生理层面

- REM duration
- REM latency
- REM density
- sleep architecture
- NREM/SWS/spindle 与 REM 的区分

### 3.3 神经机制层面

- amygdala：情绪显著性与情绪记忆相关加工
- hippocampus：情景记忆与记忆巩固相关加工
- vmPFC / dlPFC / ACC：调节、消退、控制相关加工
- insula / salience network：身体感受与显著性网络相关加工
- EEG/ERP：情绪注意、评价、错误监测或记忆提取相关成分

## 4. 写作时的证据校准

- 有 PSG 和记忆任务时，可以讨论 REM 相关睡眠过程。
- 只有总睡眠时间或睡眠质量时，不要写成 REM 特异机制。
- 只有 fMRI/ERP 情绪任务时，不要写成睡眠机制。
- 动物研究、人类实验、临床样本和健康成人样本分开综合。
- 对机制图中的每条路径标记 evidence level：direct / indirect / tentative。
