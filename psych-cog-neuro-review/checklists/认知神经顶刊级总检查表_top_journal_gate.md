# 认知神经顶刊级总检查表

## Gate 1：机制问题
- [ ] 机制问题重要、清晰、非重复。
- [ ] 机制边界、目标人群和证据类型明确。

## Gate 2：检索与筛选
- [ ] 检索式、数据库、日期和结果数可追溯。
- [ ] 纳排标准、筛选记录和排除理由透明。

## Gate 3：方法质量
- [ ] 范式、指标、预处理和统计方法可评价。
- [ ] 行为、主观、神经/生理证据分层。

## Gate 4：机制整合
- [ ] 机制矩阵完整。
- [ ] 矛盾证据和边界条件已处理。
- [ ] 每个机制主张有来源支持。

## Gate 5：投稿预备
- [ ] 来源对齐完成。
- [ ] 全角色模拟审稿完成。
- [ ] 高风险问题进入修改矩阵。
