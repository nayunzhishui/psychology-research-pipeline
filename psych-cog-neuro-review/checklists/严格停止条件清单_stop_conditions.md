# 严格停止条件清单

遇到以下情况必须停止并汇报，生成 `停止原因与补救清单_stop_reason_and_fix.md`：

- [ ] 没有检索记录，却声称完成正式综述。
- [ ] 没有全文，却做强引用或页码级来源对齐。
- [ ] 没有阅读矩阵和机制矩阵，却写机制综述正文。
- [ ] 没有方法信息，却评价 EEG/ERP、fMRI、PSG、眼动、NIRS 或心理生理证据质量。
- [ ] 没有来源对齐，却声称“可投稿”。
- [ ] 用户要求伪造脑区结果、ERP 成分、fMRI 激活、PSG 分期、眼动指标、NIRS 指标、统计结果或文献来源。

停止后只能给出补救路径，不能继续生成完成稿。
