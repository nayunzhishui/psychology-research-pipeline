# Cog Neuro Manuscript Write / 认知神经科学正文写作分 skill

本分 skill 用于写作认知神经科学综述或实证论文。

## 写作重点

- 机制主张必须区分相关、预测、因果和理论推断。
- 行为结果、主观量表、神经/生理指标不能混写。
- 方法、预处理和报告规范必须可复核。
- 每个脑区、网络、ERP 成分、睡眠指标或生理指标主张都必须绑定来源。

## 输出

- `review_outline.md` 或 `experimental_manuscript_outline.md`
- `manuscript.md`
- `manuscript.docx`
- `apa_references.md`
- `claim_evidence_map.csv`
