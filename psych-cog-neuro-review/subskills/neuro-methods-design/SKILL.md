# Neuro Methods Design / 神经与生理方法设计分 skill

本分 skill 用于选择和设计 EEG/ERP、fMRI、PSG、眼动、皮电、HRV、内分泌等方法。

## 工作内容

- 选择神经/生理模态并说明理由。
- 明确采集参数、任务同步、实验环境和质量控制。
- 明确主要指标、次要指标、排除规则和预处理计划。
- 区分行为指标、主观量表、神经/生理指标和机制解释。

## 输出

- `neuro_methods_plan.md`
- `modality_decision_log.md`
- `recording_parameters.md`
- `quality_control_plan.md`
- `reporting_plan.md`

## 禁止

不得伪造脑区、ERP 成分、fMRI 激活、PSG 分期、眼动指标或生理指标。
