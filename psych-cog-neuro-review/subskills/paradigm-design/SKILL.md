# Paradigm Design / 实验范式设计分 skill

本分 skill 用于设计或审查认知神经科学实验范式。

## 必填信息

- 任务目标和理论假设
- 刺激类型、来源、筛选和匹配标准
- trial 结构、block 结构、练习、反馈、随机化
- 反应键、时限、间隔、注视点、诱发任务
- 主要行为指标和神经/生理指标
- 排除规则和质量控制

## 输出

- `paradigm_design_plan.md`
- `stimulus_materials_log.csv`
- `behavioral_task_spec.md`
- `exclusion_rules.md`

## 质量要求

范式必须服务理论机制，不得只堆砌任务名称。所有刺激和流程要能被复现。
