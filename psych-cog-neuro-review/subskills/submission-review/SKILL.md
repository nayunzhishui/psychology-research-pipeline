# Cog Neuro Submission Review / 认知神经科学模拟审稿分 skill

本分 skill 用于认知神经科学综述、实验方案或实证论文的投稿前模拟审稿。

## 审稿维度

- 期刊 scope 与主题匹配
- 理论机制和创新性
- 范式有效性
- 神经/生理方法合理性
- 预处理和统计分析透明性
- 机制主张是否过度推断
- APA 格式、图表和参考文献
- 可重复性、数据/代码/材料可用性

## 输出

- `journal_fit_memo.md`
- `simulated_reviews.md`
- `revision_matrix.csv`
- `mechanism_claim_audit.md`
- `author_response_template.md`

## 边界

所有意见必须标注为模拟意见，不得声称来自真实期刊或真实审稿人。
