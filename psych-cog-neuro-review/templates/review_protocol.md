## 8. 检索策略原则

- 使用中英文概念块。
- 每个数据库使用适配语法，不复制同一个检索式。
- 至少保留一个宽检索式和一个精确检索式。
- 每次修改检索式都记录版本、原因、结果数量变化。
- strict 模式必须说明是否使用引文追踪、参考文献回溯、相似文献扩展。

## 11. 质量评价计划

默认采用“自定义评价 + 可选正式工具”。如果正式工具不适用，必须记录原因，而不是留空。

| 研究类型 | 自定义评价维度 | 可选正式工具 | 何时使用 | 说明 |
|---|---|---|---|---|
| 实验/随机干预研究 | 样本、随机化、任务有效性、混杂、统计报告 | RoB 2 / JBI experimental checklist | strict 或结论高度依赖该类研究 |  |
| 非随机/观察研究 | 样本、测量、混杂、统计控制、选择偏倚 | ROBINS-I / JBI observational checklist | strict 或临床/纵向证据 |  |
| 混合方法研究 | 设计一致性、数据整合、解释有效性 | MMAT | 包含质性+量化证据 |  |
| 神经影像研究 | 样本量、预处理、ROI/whole-brain、校正、多重比较 | 自定义神经科学评价 + JBI/MMAT 适配 | 几乎总是需要自定义补充 |  |
| 睡眠研究 | PSG/睡眠分期、REM 指标、时间控制、睡眠剥夺控制 | 自定义睡眠研究评价 + JBI/MMAT 适配 | REM 主题默认启用 |  |
| 动物研究 | 随机化、盲法、动物模型、样本量、转化限制 | SYRCLE | 纳入动物研究时 |  |
| 系统综述/Meta 分析 | 检索透明度、纳排、质量评价、综合方法 | AMSTAR 2 | 使用二级证据支撑重要结论时 |  |
| 综述/理论文献 | 范围、透明度、是否引用原始研究 | 自定义二级文献评价 | 背景和理论建构 |  |

## 14. AI 使用与完整性规则

- AI 可以辅助整理、提取、分类、草拟和核查。
- AI 不得编造文献、页码、DOI、PMID、CNKI ID、万方 ID、维普 ID、样本量、结果或结论。
- 未读全文不得写“全文指出”。
- 引用综述时不得把综述中的二手陈述冒充为原始研究结果。
- AI 生成的段落必须经过 `paragraph_source_alignment_matrix.csv` 核查。
