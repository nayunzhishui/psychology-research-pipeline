# Psychology Manuscript Writing / 心理学手稿写作

Write from artifacts, not memory. Literature supports the rationale and interpretation; analysis outputs supply this study's numbers.

## Workflow

1. Read the reporting plan, protocol, evidence matrix, claim map, methods plan, analysis manifest, generated results, tables, and figures.
2. Create a section-to-evidence outline before drafting.
3. Write Introduction as problem → theory → evidence/contradictions → precise gap → present study/hypotheses. Avoid citation dumps and inflated novelty.
4. Write Methods so a qualified researcher can reproduce recruitment, measures, scoring, ethics, missingness, models, software, and decisions. Match code exactly.
5. Write Results in the analysis-plan order. Report descriptive/psychometric results, primary models, secondary models, diagnostics, and sensitivities without interpretation creep.
6. Write Discussion as findings → comparison with evidence → mechanism alternatives → theoretical/practical implications → strengths/limitations → calibrated conclusion.
7. Write title and abstract last. Reconcile every abstract number with outputs.
8. Build references from verified Zotero/BibTeX metadata; preserve Zotero item key and BibTeX key distinctions.
9. Run claim-citation, citation-reference, methods-code, results-output, table/figure, terminology, reporting-checklist, and causal-language audits.
10. Revise for precision, paragraph logic, sentence variety, concision, and non-stigmatizing language. Do not optimize style by weakening technical accuracy.

## Deliverables

Write `09_manuscript/manuscript.md`, `09_manuscript/references.bib`, `09_manuscript/citation_audit.md`, and generated tables/figures. When DOCX or PDF is requested, invoke the installed document/PDF skill and complete its render-and-inspect gate.

Fail for invented citations/results, unsupported claims, methods/code mismatch, causal overstatement, omitted null findings, inconsistent sample sizes, undefined sex/gender terminology, or disclosure/ethics placeholders presented as facts.
