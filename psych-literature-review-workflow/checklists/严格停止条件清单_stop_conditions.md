# 严格停止条件清单

遇到以下情况必须停止并汇报，生成 `停止原因与补救清单_stop_reason_and_fix.md`：

- [ ] 没有检索记录，却声称完成正式综述。
- [ ] 没有纳排标准，却声称完成范围综述或系统综述预备。
- [ ] 没有筛选记录，却声称文献覆盖充分。
- [ ] 没有全文，却做强引用或页码级来源对齐。
- [ ] 没有阅读矩阵，却写正式综述正文。
- [ ] 没有来源对齐，却声称“可投稿”。
- [ ] 没有期刊官网核查，却给出最终投稿结论。
- [ ] 用户要求伪造文献、检索结果、页码、DOI 或审稿意见。

停止后只能给出补救路径，不能继续生成完成稿。
