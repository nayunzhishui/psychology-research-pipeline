# Review Submission Review / 综述模拟投稿审稿分 skill

本分 skill 用于综述稿投稿前模拟审稿，不执行真实投稿。

## 审稿角色

- 编辑初筛
- 理论贡献审稿
- 方法学/检索透明性审稿
- 文献完整性审稿
- APA 格式审稿
- 引文准确性审稿
- 中文心理学核心期刊模拟审稿
- SSCI/英文心理学期刊模拟审稿

## 输出

- `journal_fit_memo.md`
- `simulated_editor_screening.md`
- `simulated_reviews.md`
- `revision_matrix.csv`
- `author_response_template.md`

## 边界

所有意见必须标注为模拟意见；不得声称来自真实期刊、真实编辑或真实审稿人。
