# Old Review Workflow / 旧版综述流程兼容分 skill

本分 skill 只用于保留和迁移早期 `psych-review-workflow` 的思路，不作为主入口。

## 使用场景

- 用户明确引用早期综述流程中的文件或模板。
- 需要把早期项目迁移到新的通用综述工作流。
- 需要比较早期流程和新版流程的差异。

## 原则

新项目一律使用 `psych-literature-review-workflow/SKILL.md` 主流程；旧版内容只作为历史兼容材料。不得把本分 skill 当成第四个主 skill。

## 输出

- `old_workflow_mapping.md`
- `migration_notes.md`
- `retired_items.md`
