# Review Manuscript Write / 综述正文写作分 skill

本分 skill 负责基于证据矩阵写作心理学综述正文。

## 默认结构

引言与问题提出、概念界定、理论模型、测量工具、主要实证证据、方法学问题、争议与不一致结果、整合框架、研究空白、未来方向、结论。

## 写作规则

- 证据先于写作，不得先写后找文献硬配。
- 每个关键主张写作时同步绑定来源。
- APA 第 7 版用于标题层级、正文引用、参考文献、表格和图注。
- 中文优先，必要英文术语保留。

## 输出

- `review_outline.md`
- `manuscript.md`
- `manuscript.docx`
- `apa_references.md`
- `claim_evidence_map.csv`
