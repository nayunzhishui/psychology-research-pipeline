# Review Scope / 综述方向聚焦分 skill

本分 skill 只属于新的通用综述工作流，用于把宽泛研究方向聚焦为可写综述主题。

## 任务

1. 初步检索高质量综述、方法学文章和核心理论文章。
2. 识别可写方向、主题边界、目标人群、关键构念和研究空白。
3. 输出多个候选综述题目并比较可写性。
4. 帮助用户和 AI 共同确认聚焦方向。

## 输出

- `scoping_brief.md`
- `seed_reviews.csv`
- `seed_methods.csv`
- `focus_options.md`
- `导师汇报版简表.md`

## 停止条件

当至少形成 2–4 个可比较的综述方向，并明确推荐方向、备选方向和放弃理由时停止。
