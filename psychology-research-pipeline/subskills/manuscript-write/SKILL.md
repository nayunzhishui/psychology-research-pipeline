# Empirical Manuscript Write / 实证论文写作分 skill

本分 skill 负责把研究问题、文献、方法和统计结果写成 APA 兼容实证论文。

## 默认结构

- Title / 题目
- Abstract / 摘要
- Introduction / 引言
- Method / 方法
- Results / 结果
- Discussion / 讨论
- Limitations / 局限
- Conclusion / 结论
- References / 参考文献

## 写作规则

- 写作必须基于文献矩阵、方法方案和统计输出。
- 每个理论性、事实性、方法性、结果性主张都要绑定来源或数据输出。
- 不补造文献、页码、统计结果或 DOI。
- 中文正文可用 APA 第 7 版逻辑组织；参考文献保留英文规范。

## 输出

- `manuscript.md`
- `manuscript.docx`
- `apa_references.md`
- `claim_evidence_map.csv`
