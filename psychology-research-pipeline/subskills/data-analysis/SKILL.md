# Empirical Data Analysis / 实证数据分析分 skill

本分 skill 负责数据结构审计、统计分析计划执行、结果解释和偏离记录。

## 输入

SPSS、Excel、CSV、PROCESS、AMOS、Mplus、R、Python、JASP 输出或用户粘贴的统计结果。

## 工作顺序

1. 检查数据字典、变量名、反向计分、缺失值和异常值。
2. 对照冻结的 `statistical_analysis_plan.md`。
3. 执行或解释描述统计、信度、相关、回归、中介/调节、SEM、实验差异检验等。
4. 记录所有计划外分析和模型修改。
5. 生成 APA 结果文本、表格和图形建议。

## 输出

- `data_dictionary.csv`
- `data_audit.md`
- `analysis_report.md`
- `analysis_deviation_log.csv`
- `results_tables.md`
- `numeric_audit.md`

## 禁止

不得伪造数据、p 值、效应量、样本量、模型拟合指标或软件输出。
