# 顶刊级总检查表

## Gate 1：研究问题
- [ ] 研究问题清晰、重要、非重复。
- [ ] 理论贡献明确。
- [ ] 假设可检验。

## Gate 2：方法
- [ ] 设计支持推论边界。
- [ ] 样本、测量、程序和伦理可审计。
- [ ] 分析计划已冻结。

## Gate 3：数据与统计
- [ ] 数据质量已审计。
- [ ] 统计输出可追溯。
- [ ] 效应量、置信区间和稳健性检查充分。

## Gate 4：写作与引用
- [ ] 每个关键主张都有来源或数据支持。
- [ ] 引用与参考文献一致。
- [ ] 讨论没有过度推断。

## Gate 5：开放科学与投稿
- [ ] 数据、代码、材料、协议和限制说明明确。
- [ ] 已完成全角色模拟审稿。
- [ ] 高风险问题已进入修改矩阵。

结论：通过 / 待补充 / 暂不建议投稿
