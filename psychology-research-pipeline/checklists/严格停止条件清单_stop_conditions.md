# 严格停止条件清单

遇到以下情况必须停止并汇报，生成 `停止原因与补救清单_stop_reason_and_fix.md`：

- [ ] 没有数据，却要写结果。
- [ ] 没有统计输出，却要写 p 值、效应量、置信区间或模型拟合指标。
- [ ] 没有全文，却要做强引用或页码级来源对齐。
- [ ] 没有量表来源，却要写“采用某量表”。
- [ ] 没有伦理信息，却要声称伦理合规。
- [ ] 没有期刊官网核查，却要给出最终投稿结论。
- [ ] 没有来源对齐，却要声称“可投稿”。
- [ ] 用户要求伪造数据、文献、统计结果、伦理审批或审稿意见。

停止后只能给出补救路径，不能继续生成完成稿。
